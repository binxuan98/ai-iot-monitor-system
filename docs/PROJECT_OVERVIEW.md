# 项目总体说明

## 项目定位

《智慧环境监测与异常预警平台》是一个教学型开源项目，目标是帮助高职学生在真实业务场景中完成 AI+IoT 系统的设计、编码、测试与协作交付。

## 教学目标

1. 理解前后端分离开发流程  
2. 掌握 Flask 接口开发与 SQLite 数据管理  
3. 理解规则检测与评分式智能分析  
4. 能进行 GitHub 团队协作开发  
5. 能完成项目文档编写与演示答辩

## 功能模块

- 用户认证模块（登录/退出）
- 环境数据模块（最新数据/历史数据）
- 异常检测模块（阈值规则）
- 智能分析模块（评分+状态说明）
- 告警记录模块（历史异常）
- 数据可视化模块（折线图/柱状图/仪表盘）
- 系统日志模块（关键行为记录）

## 适配场景

- 实训室
- 小型机房
- 智慧教室
- 校园环境监测
- 智慧农业教学模拟场景
