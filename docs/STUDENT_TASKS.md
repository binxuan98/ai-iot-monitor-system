# 学生岗位任务说明

## 一、项目经理

- 维护需求清单、里程碑和任务看板
- 组织每日站会，跟踪阻塞问题
- 协调跨角色联调与发布节奏
- 验收每个迭代目标并输出总结

## 二、前端开发工程师

- 负责登录页与仪表盘页面实现
- 对接后端接口并处理异常提示
- 完成图表展示与表格渲染
- 做基础响应式适配

## 三、后端开发工程师

- 编写 Flask API
- 完成 SQLite 表结构与数据读写
- 实现告警与日志记录机制
- 保证接口格式稳定并可测试

## 四、算法工程师

- 维护阈值规则及评分策略
- 设计环境状态说明文案规则
- 对异常检测结果进行调参优化
- 扩展简单预测或趋势分析功能

## 五、测试工程师

- 编写并维护接口测试
- 输出测试报告与缺陷列表
- 组织联调测试与回归测试
- 复现并验证缺陷修复结果

## 六、运维/文档工程师

- 编写启动脚本与运行指南
- 维护 README 与接口文档
- 准备课堂演示环境
- 收集版本变更记录与发布说明
